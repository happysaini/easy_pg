Easy PG Web Application

Project Starts : 20 April , 2019
This Project is an initiative to reduce the burden of finding rooms as paying guest.

EasyPG is a web application which serve the main purpose of helping both Owner and Tenant. Owner can post the Details of room or house online from where tenant can approach him or Tenants can also post request to web application for getting suitable place in particular area from where owner can approach him.
